# La Huella de un Amor Breve 🦋

Este repositorio contiene el código fuente de la página web diseñada para **"La Huella de un Amor Breve"**, una guía de acompañamiento creada para transitar el duelo gestacional con ternura y respeto.

## 🌟 Descripción del Proyecto
La página es una *Landing Page* enfocada en ofrecer información clara, brindar acompañamiento y facilitar la adquisición de la guía de forma segura. El diseño busca transmitir paz, empatía y profesionalismo.

## 🛠️ Tecnologías Utilizadas
- **HTML5:** Estructura semántica del contenido.
- **CSS3:** Estilos responsivos, diseño adaptativo (Media Queries) y animaciones personalizadas (efecto heartbeat).
- **JavaScript:** Lógica de interacción para los modales de compra e índice, y gestión de estados de interfaz.

## ✨ Características Principales
- **Diseño Responsivo:** Adaptación completa a dispositivos móviles y escritorio.
- **Experiencia de Usuario (UX):** Interacciones fluidas mediante modales de compra.
- **Accesibilidad:** Estructura de navegación clara y enlaces de contacto directos.
- **Botón Flotante:** Acceso rápido a WhatsApp para consultas, con optimización para que no obstruya el contenido.
- **Optimización:** Código optimizado para una carga rápida y fluida.

## 🚀 Cómo visualizar el proyecto
Este proyecto es una aplicación estática. Puedes visualizarlo directamente clonando este repositorio o abriendo el archivo `index.html` en tu navegador.
